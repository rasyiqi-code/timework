## Pendahuluan

Timework adalah aplikasi manajemen proyek yang dirancang untuk membantu tim berkolaborasi, melacak kemajuan tugas, dan mengelola alur kerja proyek dengan terstruktur.

## Memulai

### Akses Aplikasi
Buka aplikasi melalui browser web pilihan Anda. Pastikan Anda terhubung ke internet.

### Login dan Registrasi
Halaman pertama yang Anda temui adalah halaman Login.
- **Login**: Masukkan kredensial akun Anda (email dan password) untuk masuk.
- **Registrasi**: Jika belum memiliki akun, gunakan opsi pendaftaran untuk membuat akun baru. Ikuti proses onboarding untuk melengkapi profil Anda.

## Dashboard
Setelah berhasil masuk, Anda akan diarahkan ke halaman Dashboard. Halaman ini memberikan ringkasan aktivitas dan navigasi cepat ke fitur-fitur utama:
- Sidebar di sebelah kiri berisi menu navigasi utama (Projects, My Tasks, Settings, dll).
- Area utama menampilkan ringkasan atau widget penting (tergantung konfigurasi).

## Manajemen Proyek

### Membuat Proyek Baru
1. Masuk ke menu **Projects**.
2. Klik tombol **"Create Project"** (atau ikon "+").
3. Sebuah modal akan muncul. Isi detail proyek:
   - **Nama Proyek**: Judul untuk proyek Anda.
   - **Protocol**: Pilih protokol yang sesuai (Wajib). Protokol menentukan struktur atau alur kerja dasar proyek.
   - **Detail Tambahan**: Isi field lain yang mungkin muncul tergantung pada konfigurasi sistem (misal: client, kategori).
4. Klik **Create** untuk membuat proyek.

### Melihat Daftar Proyek
Di halaman **Projects**, Anda akan melihat daftar semua proyek. Klik pada kartu proyek untuk masuk ke dalamnya.

### Detail Proyek (View Board & Table)
Dalam proyek, Anda dapat beralih tampilan:
- **Board View**: Tampilan kanban (kolom status). Geser (drag-and-drop) kartu tugas untuk mengubah statusnya.
- **Table View**: Tampilan baris untuk melihat detail banyak tugas sekaligus.

### Mengelola Item Tugas
Di dalam proyek, Anda dapat membuat tugas baru:
1. Klik tombol tambah item pada kolom yang diinginkan.
2. Isi judul dan deskripsi tugas.
3. Tugaskan anggota tim menggunakan **Assignee Selector**.

## Tugas Saya (My Tasks)
Menu **My Tasks** adalah pusat notifikasi personal Anda. Halaman ini menampilkan **semua tugas yang ditugaskan kepada Anda** dari berbagai proyek dalam satu tempat.
- Gunakan halaman ini untuk memantau prioritas harian Anda.
- Klik tombol **"Open Project"** pada kartu tugas untuk langsung menuju ke halaman proyek terkait dan melihat konteks tugas tersebut lebih detail.

## Pengaturan

### Profil & Akun
Akses menu **Account Settings** untuk mengelola informasi pribadi Anda, mengubah password, atau memperbarui foto profil.

### Pengaturan Formulir Proyek
Menu **Settings > Project Form** memungkinkan Anda (jika memiliki akses) untuk menyesuaikan template atau struktur data yang digunakan dalam proyek.

## Fitur Admin

### Manajemen Protokol
*(Hanya untuk pengguna dengan akses Admin)*
Menu **Admin > Protocols** digunakan untuk mengelola "Protokol". Protokol kemungkinan besar adalah templat standar atau prosedur operasi (SOP) yang dapat diterapkan pada proyek atau tugas untuk memastikan konsistensi alur kerja (Misal: Protokol KBM).
- Admin dapat membuat, mengedit, dan menghapus protokol.
- Item dalam protokol dapat disusun menggunakan **Item Builder**.
